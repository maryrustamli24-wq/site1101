C++ Hotel Booking System

I developed a fully interactive hotel booking system in C++ that allows users to search, select, and book hotels while automatically calculating the final bill based on stay duration, room type, number of guests, and hotel prices.

Key Features:

  • User and admin login system
  • Date selection + calculation of total days & nights
  • Multiple hotel categories (Business, Resort, Boutique)
  • Room selection logic based on number of guests
  • Automatic bill calculation
  • Sorting hotels by price
  • Admin panel for modifying hotel data (add, delete, search, update)
  • Colorful UI using Windows console colors
  • ASCII-art logo interface
  • Error handling and input validation

Technologies Used:

  • C++
  • Windows.h for console UI
  • Arrays & functions
  • Procedural logic and date calculations

This project demonstrates my proficiency in C++ programming, system design, and creating user-friendly console applications with comprehensive functionality including authentication, data management, and automated calculations.

This is a local C++ project developed for academic purposes.
