How Does Hardware Work?

In this project, we explored how digital logic works at the hardware level by building OR and NAND gates on a breadboard. Using transistors, resistors, wires, LEDs, and buttons, we created working circuits that respond to different input combinations. By testing each configuration, we saw how simple electronic components can be combined to implement logical operations used inside real computers.

Components Used:

  • Breadboard and jumper wires
  • Transistors and resistors
  • LEDs and push buttons
  • Power supply (battery or adapter)

This project helped us understand the connection between abstract logic gates and physical hardware. We learned how voltage levels represent binary states, how circuits are debugged with LEDs as indicators, and how hardware fundamentals form the basis for more advanced computer architecture.

Hour of Code (AI)

We organized an Hour of Code (AI) event at secondary school No. 318, located in Garachukhur settlement, Surakhani district, Baku city to introduce children to the world of programming. During the workshop, we explained basic algorithms, practiced simple Python operations, and showed how computers follow step-by-step instructions. The goal was to make coding fun, accessible, and inspiring for students who were experiencing programming for the first time.

Activities and Technologies:

  • Teaching basic algorithms and logical thinking
  • Hands-on practice with simple Python operations
  • Playing Microsoft MakeCode Bug Arena to explore coding through games
  • Presenting participation certificates to students

Through this project, we learned how to communicate technical concepts to younger students and how to design an engaging learning experience that mixes theory, practice, and play. It strengthened our teaching, communication, and leadership skills while promoting digital literacy in the community.

C++ Hotel Booking System

I developed a fully interactive hotel booking system in C++ that allows users to search, select, and book hotels while automatically calculating the final bill based on stay duration, room type, number of guests, and hotel prices.

Key Features and Components:

  • User and admin login system
  • Date selection and calculation of total days and nights
  • Multiple hotel categories (Business, Resort, Boutique)
  • Room selection logic based on number of guests
  • Automatic bill calculation and sorting hotels by price
  • Admin panel for adding, deleting, searching, and updating hotel data
  • Colorful console UI with ASCII-art logo and input validation

Technologies Used:

  • C++
  • Windows.h for console UI
  • Functions, arrays, and procedural logic

This is a local C++ project developed for academic purposes.
