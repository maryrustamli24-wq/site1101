# Images Folder

## Required Images

Please add the following images to this folder:

1. **profile-photo.jpg** (or .png)
   - Used on: Home page (index.html)
   - Recommended size: 500x500 pixels (square)
   - Place your profile photo here

2. **project1.jpg** (or .png)
   - Used on: Projects page (projects.html)
   - Recommended size: 1200x600 pixels (landscape)
   - Place your project screenshot/image here

## Image Format Support
- .jpg / .jpeg
- .png
- .gif

## Tips
- Use descriptive filenames (no spaces, use hyphens)
- Compress large images for better website performance
- Keep file sizes under 1MB for faster loading
